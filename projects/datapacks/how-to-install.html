How to install a datapack for Minecraft Java Edition

Installing a datapack on a new world

Go to the "Create New World" screen, then to "Data packs". Then click "Open Pack Folder". Folder". Navigate to "datapacks". Now download the datapack and open your "downloads" folder. Drag the datapack into the datapacks folder. If it is a zip folder you need to open it. If theres a "data" folder and a "pack.mcmeta" file you can just move the .zip to the Datapack folder. If not move the folder insite into the Datapack folder.

Installing a datapack on a existing world

First you need to go to the world files. Open Mincecraft, go to Singleplayer, Highlight the world you want to add the datapack to, then click "Edit". Now select "Open World Folder". Navigate to "datapacks". Now download the datapack and open your "downloads" folder. Drag the datapack into the minecraft world folder named "datapacks". If it is a zip folder you need to open it. If theres a "data" folder and a "pack.mcmeta" file you can just move the .zip to the Minecraft world. If not move the folder insite into the Minecraft world datapack folder.
